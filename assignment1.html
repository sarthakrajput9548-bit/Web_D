HTML Assignment

Question & Answer

Q1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages.

Q2. What is a web browser?

A web browser is software used to access and display web pages, such as Chrome or Firefox.

Q3. What is a tag in HTML?

A tag is a keyword enclosed in angle brackets used to define elements in HTML.

Q4. What is the use of <h1> tag?

The <h1> tag is used to define the main heading of a web page.

Q5. What is the use of <p> tag?

The <p> tag is used to define a paragraph in HTML.

Q6. What is an attribute?

An attribute provides additional information about an HTML element.

Q7. What is the use of <br> tag?

The <br> tag is used to insert a line break.

Q8. What is the use of <a> tag?

The <a> tag is used to create hyperlinks.

Q9. What is a list?

A list is used to group related items together. HTML supports ordered and unordered lists.

Q10. What is the use of <img> tag?

The <img> tag is used to display images on a web page.
