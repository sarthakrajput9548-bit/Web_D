Introduction

HTML stands for HyperText Markup Language. It is used to create the structure of web pages using different tags and elements.

Features of HTML

  • Easy to learn and use
  • Platform independent
  • Supports text, images and links
  • Helps in creating web pages

Basic HTML Tags

  1. <html>
  2. <head>
  3. <title>
  4. <body>
  5. <p>

Conclusion

HTML is the basic and most important language for web development. Learning HTML is the first step to create websites.
